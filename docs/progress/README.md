
# Progress

## Phase 1 — 8 June 2026

This folder contains the project progress, organized phase by phase.

### Phase 1
- Initial prototype based on ESP32-C3 SuperMini nodes.
- Node-to-node communication through ESP-NOW.
- Dynamic roles: Master, Slave, and Candidate.
- Heartbeat mechanism and Master election.
- Sensor reading and local and remote visualization.
- Web GUI for monitoring and configuration.
- Parameters stored in NVS.
- Prototype still under breadboard development.

### Notes
- At the moment, only Phase 1 is available.
